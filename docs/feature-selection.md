# CareFlow MVP 기능 선별 메모

## 이번 저장소에서 남긴 것
- 전국 장기요양기관 지도
- 센터 검색 / 상세 페이지
- 공개 생활 피드
- 기관 Claim 요청
- 상담 요청
- 기관 피드 작성
- 피드 공개 범위(전체 공개 / 보호자 공개 / 직원 공개) 데이터 구조

## 이번 저장소에서 의도만 남기고 뺀 것
- 댓글
- 반응 / 좋아요
- 인기 정렬
- DM / 메시지
- 관리자 승인 화면
- 사고 로그 공개 지표
- about / faq / guide / qna 등 보조 페이지

## 반영 원칙
- 큰 버전의 철학(보호자 신뢰, 생활 피드, 기관 Claim)은 유지
- rebuild 버전의 단순한 구조를 기준 저장소로 사용
- 본업과 병행 가능한 범위까지만 1차 배포 대상에 포함
